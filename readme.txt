#	
#	封装了与 JSON 相关的常用的操作。包括对象的序列化和反序列化以及写出 JSON 串内容。
#	
#	序列化对象时支持过滤对象的类型属性和字段名称属性。
#	
#	过滤类型属性以防止对象与对象之间存在循环引用引发无限循环序列化导致发生异常。
#	
#	过滤字段名称属性以防止不必要的字段属性被序列化。
#	
#	项目依赖 gson.jar 包。JsonHttpUtil 由于依赖 servlet api，因此将它单独抽了出来。
#	
#	开发环境：eclipse（indigo） + maven
#	
#	eclipse 导入项目：
#	
#	Import ——> Maven ——> Existing Maven Projects
#	
#	Ant 任务：
#	
#	m-package ( 打包 )
#	
#	m-clean   ( 清除 )
#	
#	m-idea    ( 生成 idea 项目 )
#	
#	m-eclipse ( 生成 eclipse 项目 )
#	
#	demo 项目地址：https://github.com/fancores/json-demo
#
